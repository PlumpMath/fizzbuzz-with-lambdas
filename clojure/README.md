# Clojure Lambda FizzBuzz

Functional implementation of
[FizzBuzz](http://en.wikipedia.org/wiki/Fizz_buzz) using Clojure 

### Requires
[leiningen](http://leiningen.org)

### To run 

```
$ lein uberjar
$ ./run.sh 42
```

Or

```
$ lein run 42
```
