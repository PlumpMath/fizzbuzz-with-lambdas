# Scala Lambda FizzBuzz

Functional implementation of
[FizzBuzz](http://en.wikipedia.org/wiki/Fizz_buzz) using Scala

### Requires
[sbt](http://www.scala-sbt.org)


### To run 

```
$ sbt package
$ ./run.sh 42
```

Or

```
$ sbt
sbt> run 42
```
